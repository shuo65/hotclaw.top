📈 排行榜数据维度

热门Claw排行榜基于多维度数据综合计算，每周自动更新（人工审核后发布）：

⭐
GitHub Stars

项目收藏数，反映社区认可度

📥
下载量

NPM下载数，实际使用频率

💬
社区讨论

Issues、Discord活跃度

🔄
更新频率

项目维护活跃程度

📚
文档质量

文档完整度与可读性

🎯
适用场景

功能匹配度评分

🔍 数据来源

排行榜数据来自以下权威渠道，确保信息准确可靠：

  • GitHub API: 获取Stars、Forks、Issues数量
  • NPM Registry: 查询包下载量、版本号
  • OpenClaw官方文档: 功能特性、配置参数
  • 社区反馈: Discord、中文社区讨论热度
⚠️ 注意： 数据每周日自动采集，人工审核后发布，确保数据真实有效。

⚙️ 排名算法（简化版）

综合得分计算公式：

Score = 
  (Stars × 0.25) + 
  (Downloads × 0.30) + 
  (Activity × 0.20) + 
  (Documentation × 0.15) + 
  (Recency × 0.10)

* 各项评分归一化处理，满分100分。
* Activity包含近期commits、issue响应速度。
* Recency考虑最近更新时间（越新权重越高）。
